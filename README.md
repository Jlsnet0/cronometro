# Cronômetro

Aplicação de cronômetro desenvolvida com **HTML, CSS e JavaScript**, permitindo iniciar, pausar e zerar.

---

## Funcionalidades

* Iniciar contagem do tempo
* Pausar cronômetro
* Zerar contagem
* Atualização dinâmica na tela

---

## Tecnologias

* HTML5
* CSS3
* JavaScript
